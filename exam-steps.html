Github for the Exam

Using the github.com Web Interface - create a new repository

Setting up a new repository

  • From your github.com home page click "Repositories" (near the top center).
  • Click the green "New" button near the upper right corner.
  • Enter the name and description of the new repository. (ie. bootCampExam
  • Make the new repository Public or Private
  • Click the green "Create Repository" button at the bottom.

Using terminal or git bash - Update the Exam Code

Cloning a Repository - use a new terminal/gitbash window

  • On your PC change to a high level directory where your git projects with be stored (ie. "cd" - with no parameter, puts you in your home directory.
  • Type "git init" to setup the files Git will need to manage files on your PC.
  • Type "git clone git@github.com:HackerUSA-CE/placement-exam-starter-code.git" to clone the exam repository.
  • Type "cd placement-exam-starter-code
  • Type "git status" to confirm git knows about your files.
  • Type "code ." to start VS Code editor and open the directory you just cloned.
  • Modify the code per exam instructions.
  • Type "git status" to see that git has acknowledged your changes and git prompts you to the next step(s).
  • Type "git add" to make git aware of your changes.
  • Type "git commit -m "Your comments describing all your changes so the approver will know what changes to look for."
  • Type "git remote set-url origin https://github.com/<your github name>/bootCampExam.git" to point to bootCampExam, under your github name. The origin was set to "origin" with the "git clone" command and
  • Type "git push origin main" to send your changes to your github web repository. Since we did not create a branch, you will send your changes to the "main" branch. Your code changes should now be viewable on github (the web tool.) Be sure to check for errors that would prevent your repository from being updated.
